# Servants
## Description

Retrieve accounts which are servants of a given account
## Info

**Command**

    cleos get servants

**Output**

```
Usage: cleos get servants account

Positionals:
  account TEXT                The name of the controlling account
```

## Command

    cleos get servants inita

## Output

```
{
  "controlled_accounts": [
    "tester"
  ]
}
```
